Tommis web app test

Det här är en test miljö för olika enkla mobil appar kodade med hjälp av AI efter anvisningar av Tommi Villenheimo.

Det finns ingen djupare tanke bakom Github kontot, det var helt enkelt det enklaste sättet att kunna använda egna körbara web appar på iPhone och android mobila enheter där hög säkerhet förhindrar körning av icke säkerhets certifierad web kod direkt från enhetens fil utrymme.
